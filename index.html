splose widget shape generator

A body rectangle with two independent diagonal kicks. Each kick can be dialed to zero — set both to zero for a plain rounded rectangle, or one to zero for a single chamfer. Every angled edge is locked to the chosen angle; only corners are rounded. Fully responsive via viewBox.
